# 🎉 Como Contribuir para o build-your-github-repository

Antes de mais nada, muito obrigado por dedicar seu tempo para contribuir! 💙 

Este projeto foi criado para ajudar iniciantes a dominarem o GitHub. Portanto, **toda contribuição é bem-vinda**, desde a correção de um erro de digitação (typo) até a criação de um novo template de README. Não tenha medo de errar, estamos todos aprendendo!

## 🐛 Como reportar um problema (Bug) ou sugerir uma ideia

Se você encontrou um link quebrado, um erro nas instruções ou tem uma ideia para um novo template:
1. Vá até a aba **Issues** do repositório.
2. Verifique se alguém já não reportou o mesmo problema.
3. Clique em **New Issue** e descreva o seu problema ou ideia com o máximo de detalhes possível.

## 💻 Passo a passo para enviar o seu código (Pull Request)

Se você quer colocar a mão na massa e alterar o código ou adicionar um template novo, siga este passo a passo padrão do GitHub:

1. **Faça um Fork do projeto:** Clique no botão "Fork" no canto superior direito desta página para criar uma cópia deste repositório na sua conta.
2. **Clone o repositório:** Faça o clone do seu fork para o seu computador executando:
`git clone https://github.com/SEU-USUARIO/build-your-github-repository.git`
3. **Crie uma Branch:** Crie uma ramificação (branch) para a sua alteração com um nome claro executando:
`git checkout -b feature/novo-template-perfil`
4. **Faça suas alterações e o Commit:** Faça as mudanças necessárias e salve-as usando os Commits Convencionais executando:
`git commit -m "docs: adiciona template de perfil focado em backend"`
5. **Envie para o seu Fork (Push):** Envie o código para o seu repositório executando:
`git push origin feature/novo-template-perfil`
6. **Abra um Pull Request (PR):** Volte para a página original do repositório no GitHub e você verá um botão verde sugerindo a criação do Pull Request. Clique nele, preencha as informações do que você alterou e envie!
